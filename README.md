## Sales Data Analysis Project

This project focuses on analyzing retail sales data to uncover meaningful insights and trends. It is the end-to-end data analysis using Python that include data cleaning, visualization. The goal is to demonstrate the full data analysis process, including data cleaning, exploration, visualization, and interpretation.

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Analyze sales performance across categories and regions
- Identify trends over time
- Generate actionable business insights
  

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- 

## 📂 Dataset
The dataset used in this project is a retail sales dataset obtained from Kaggle.


## 🔍 Key Steps

### 1. Data Cleaning
- Removed duplicate records  
- Handled missing values  
- Converted date columns to proper format  

### 2. Exploratory Data Analysis (EDA)
- Analyzed total sales performance  
- Compared sales by category and region  
- Examined monthly sales trends  

### 3. Data Visualization
- Bar charts for category and regional sales  
- Line chart for monthly trends  


## 📈 Key Insights
- The Technology category generated the highest sales (~$827455.87), highlighting its dominance compared to other categories.
- The West region recorded the highest overall sales, suggesting it is the most profitable market
- Sales showed a peak during November and Dicember likely due to seasonal promotion and holiday demand.
